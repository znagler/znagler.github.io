Knowing nothing about you, the predictor chooses Sanders with a probability of 25.1%

Can attributes like first name and occupation reveal which presidential candidate you prefer? Around 900,000 rows of data from the Federal Election Commission says the answer is yes, fairly well. This app uses a trained random forest classifier to predict a candidate given some info, and on a cross-validated test set it's accurate over 60% of the time.

Technologies used: python (sci-kit learn, pandas, flask), javascript (d3.js, jQuery) and semantic-ui.
