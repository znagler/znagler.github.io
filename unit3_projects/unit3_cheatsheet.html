Basic Regex Cheatsheet

The string... Scanning Regex... Yields... Because...
"apple" /ap/ ["ap"] 1 instance of "ap"
"grape apple" /ap/ ["ap", "ap"] 2 instances of "ap"
"grape apple" /g./ ["gr"] . is any char
"grape apple" /g.*/ ["grape apple"] .* is any amt of any char
"grape apple" /p+/ ["p","pp"] + is one or more of given char, consecutively
"grape apple" /\s/ [" "] \s is space
"Nintendo 64" /[a-z]/ ["i", "n", "t", "e", "n", "d", "o"] [a-z] accepts any lowercase letter
"Nintendo 64" /[A-Za-z] ["N", "i", "n", "t", "e", "n", "d", "o"] [A-Za-z] accepts any letter
"Nintendo 64" /[A-Za-z]+/ ["Nintendo"] given char for + is any letter
"Nintendo 64" /[a-zA-Z]{3}> ["Nin", "ten"] {3} allows 3 of given char
"Nintendo 64" /[a-zA-Z]{2,3} ["Nin", "ten","do"] {2,3} allows 2 or 3 of given char
"Nintendo 64" /^M/ [] with ^ only searches front of string
"Nintendo 64" /^N/ ["N"] with ^ only searches front of string
"Nintendo 64" /endo$/ [] with $ only searches end of string
"Nintendo 64" /endo 64$/ ["do 64"] with $ only searches end of string
"Nintendo 64" /0-9]+ ["64"] [0-9] accepts any single digit
"color" /colou?r/ ["color"] ? allows 0 or 1 instance of given char
"colour" /colou?r/ ["colour"] ? allows 0 or 1 instance of given char
"gray" /gr[ae]y/ ["gray"] accepts any one char in closed brackets
"grey" /gr[ae]y ["grey"] accepts any one char in closed brackets
"$99.00" /\$99\.00/ ["$99.00"] \ escapes given char so is needed for special chars
