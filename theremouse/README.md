# [theremouse](https://znagler.github.io/theremouse/spotify.html)

## The Project
Theremouse is a single page app built with JavaScript, jQuery, and Firebase that lets you play the [theremin](http://en.wikipedia.org/wiki/Theremin) with your mouse while other users join in real time.
