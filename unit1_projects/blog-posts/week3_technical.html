Why JavaScript?

Despite its complicated history and lifetime of being confused with Java, the lightweight, object-oriented language JavaScript has emerged as the universally-used programming language for the Web.

JavaScript is most commonly known as the language that builds the highly interactive applications we all love on the Web, but its reach extends far beyond browsers. JavaScript is also used on:

  • Operating Systems
  • Databasaes
  • Servers
  • Mobile Devices
  • Televisions and other electronics

Pros:

  • One of the only reliable ways to do client-side Web programming
  • Since it runs client-side, it reduces load on servers which is key for high-traffic sites
  • Can be inserted into any Web page regardless of file extension
  • Relatively easy to learn, using common-sense English words and sharing syntax with other popular languages
  • Interpreted, not compiled

Cons:

  • Rendering can vary between browsers
  • Vulnerable to some security issues
  • Client must enable JavaScript

While JavaScript has its weaknesses, its future looks bright because of the increasing demand for applications of all types to run on browsers and mobile devices. Since the server sides is also now embracing JavaScript with tools like Node.js, JavaScript is one of the most useful languages a programmer can learn, and will stay that way for at least the near future.
