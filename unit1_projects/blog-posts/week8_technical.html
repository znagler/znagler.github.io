Regex

Regex is a language used for selecting and modifying text. It uses a series of operators to be able to perform very specific and complex searches. It is most often used in conjunction with some other language, and luckily, it is built into common ones like Javascript and Ruby.

To try it in IRB, simply type in a string and call the scan method, and the Regex will go in the argument. For example, we could have "apple".scan(/ap/). This is going to look through the word apple, and try to find 'ap,' and if it does, it will return it. If it doesn't, it returns an empty array, and if it finds it twice it will return it twice.

Regex uses special characters like . , * , + and many more to allow for different types of searches. A period, for exmaple, means one of any character, so if I searched on the string "the cute cat in the cottage" for /c.t/, the returning array would be ["cut","cat","cot"] since all those strings match.

Check out a few more examples on my Basic Regex Cheatsheet

A few real-world applications of Regex are finding valid addresses and phone numbers, finding (and maybe swapping) certain text in the HTML of a page, URL and search term parsing, and being able to understand this t-shirt.
